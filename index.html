Aufgaben zu Higher-Order Functions in JavaScript

05/2022 Steffen Balmer

Description:

Funktionen höherer Ordnung in JavaScript sind eine spezielle Kategorie von Funktionen, die entweder Funktionen als Argument akzeptieren oder Funktionen zurückgeben. Funktionen höherer Ordnung bieten den Vorteil der Wiederverwendbarkeit: Das Hauptverhalten wird von der Funktion höherer Ordnung selbst bereitgestellt, und indem Sie eine Funktion als Argument akzeptieren, erweitern Sie dieses Verhalten nach Belieben.

Zum besseren Verständnis eine Website die den Unterschied zwischen Higher-Order- und regulären Funktionen: [LINK]

Funktionen höherer Ordnung ermöglichen es uns, über Aktionen zu abstrahieren, nicht nur über Werte. Sie kommen in mehreren Formen vor. Beispielsweise können wir Funktionen haben, die neue Funktionen erstellen.

Um High-Order-Funktionen und reguläre Schleifen-Funktionen besser zu verstehen, werden wir im Folgenden mehre Übungsaufgaben durchzuführen. Gegeben sind High-Order-Funktionen, die in mögliche Statements für Schleifen umgewandelt werden sollen. Mögliche Schleifen-Statements sind:

  • for ([initialerAusdruck]; [Bedingung]; [erhöhenderAusdruck])
  • do... while ([Bedingung])
  • while ([Bedingung])
  • for...in: for (variable in object) { }
  • for...of: for (variable of object) { }
  • label: anweisung;
  • break: [label];
  • continue [label];
